# Personal Library Manager

A Python CLI application to manage a personal book library — built as a final project for the **HTUX Coding Like a Pro** course (part of the Python Power-Up pathway).

## About

This is a command-line application where users can manage their personal book collection. The project was built to apply core Python concepts learned throughout the course, from basic functions all the way to file handling and error management.

## Features

- Add books with title, author, and category
- Save library to a JSON file for persistent storage
- Load library from a saved JSON file
- Visualize books by category using a text-based histogram

## Concepts Practiced

- **Procedural Programming** — functions, loops, and conditional logic
- **File Operations** — reading and writing JSON files
- **Error Handling** — using `try/except` for `FileNotFoundError` and `JSONDecodeError`
- **Data Structures** — dictionaries and lists

## How to Run

```bash
python main.py
```
